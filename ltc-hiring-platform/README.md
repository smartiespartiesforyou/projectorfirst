# LTC Hiring Platform (Phase 1)

A calm, focused hiring experience for long-term care organizations and applicants.

## Built in Phase 1

- Homepage
- Jobs list page
- Job detail page
- Fit-check prompts
- Application form UI
- Supabase SQL schema

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Stack

- Next.js + React
- Tailwind CSS
- Supabase (schema in `db/schema.sql`)
